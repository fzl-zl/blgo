笔记 

form必须要有action属性     action表示提交的服务器地址                                                                                                                      
< input type="" name="">     type可选值：text             文本框
                                       password         密码框
                                       radio            单选框    必须要有相同的name属性 必须指定value属性  checked属性可以将单选按钮设置为默认选中
                                       checkbox         复选框    必须要有相同的name属性 必须指定value属性  checked属性可以将单选按钮设置为默认选中
                                       submit           提交按钮
                                       button           按钮
                                       reset            重置按钮
                                       search           搜索框
< button type=""></button >    type可选值:
                                        submit
                                        button
                                        reset        下面这种按钮的方式使用的多  button有闭标签 可在其中插入其他东西  input是自结束标签
 select           下拉列表  必须要有相同的name属性 必须指定value属性  option标签表示选项            
autocomplete="off" 关闭自动补全
readonly  将表单设置为只读 数据会提交
diaable   将表单设置为禁用 数据不会提交
autofocus 将表单设置为自动获取焦点 （光标直接会在获取焦点的框）
网站图标一般都存储在网站的根目录下 名字一般都叫 favicon.ico
transition：过渡  
transition-property:    指定要执行过度的属性（如width ，height ，color， all， margin-left...） 多个属性之间用逗号，隔开 大部分属性都支持过渡效果
transition-duration：    指定过度要持续的时间
transition-timing-function: 过渡的时序函数    可选值：  ease           默认值  慢速开始 先加速 后减速
                                                      linear         匀速运动
                                                      ease-in        加速运动
                                                      ease-out       减速运动
                                                      ease-in-out     先加速 后减速
                                                      cubic-bezier(1,1,1,1)     贝塞尔曲线  https://cubic-bezier.com/#.17,.67,.83,.67
                                                      steps(n,end)        默认值是end  可以不写  在结束的时候跳步
                                                      steps(n,star)       在开始的时候跳步
transition-delay：延时
过渡简写:   transition:执行时间 执行属性 延时...; 执行时间和延时两者中  执行时间必须写前面 
动画 animation  设置动画时首先要定义关键帧  然后再在css中用 animation-name：name；使用动画
                                   @keyframess name{    name是动画名字 
                                              from{       from是指定开始位置的关键帧 from也可写成0%
                                                     margin-left：0；
                                              }
                                              to{         to是指定结束位置的关键帧   to可写成100%
                                                      margin-left：100Px；
                                              }

                                                   }
animation-duration：动画执行时间
animation-delay：延时
animation-timing-function：和过渡一样  时序函数
animation-iteration-count：动画执行次数  infinite代表执行无限次
animation-direction：指定动画运行的方向  可选值：normal  默认值  从from向to执行 每次都是这样
                                               reverse  从to向from执行 每次都是这样
                                               alternate  从from向to执行 重复执行动画时反向执行
                                               alternate-reverse 从to向from执行 重复执行动画时反向执行
animation-play-state：设置动画执行状态  可选值：running   pause
animation-fill-mode：动画填充模式   可选值：none  默认值 动画执行完毕回到原来位置  在延时的时候元素不会有变化
                                          forward  动画执行完毕停止在动画结束的位置
                                          backward  动画延时等待  元素会处于开始位置  元素会有变化 延时的过程中元素会变成关键帧里from或者to的状态
                                          both   结合forward和backward
